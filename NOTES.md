# 메모

## Claude Code 커스텀 에이전트 공유 방법

### 핵심
- 데스크탑/폰/웹은 각자 **다른 환경**이라, 데스크탑의 `~/.claude/agents/`는 폰·웹 세션에 자동 동기화되지 않음
- 어디서든 같은 에이전트를 쓰려면 **프로젝트 저장소의 `.claude/agents/`에 커밋**해야 함

### 위치별 차이
| 위치 | 범위 | 동기화 |
|---|---|---|
| `~/.claude/agents/<이름>.md` | 개인 전용 (해당 기기) | ❌ 다른 기기/세션에 안 보임 |
| `.claude/agents/<이름>.md` (프로젝트) | 이 저장소를 쓰는 모두 | ✅ git push/pull로 공유 |

### 권장 흐름
1. 데스크탑에서 `~/.claude/agents/` 안의 에이전트 정의 파일을 확인
2. 같은 내용을 이 저장소의 `.claude/agents/<이름>.md`로 복사
3. `git add .claude && git commit && git push`
4. 이후 폰·웹 세션에서도 자동으로 인식됨, 팀원들도 `git pull`로 동일하게 사용

### 주의
- 공개 저장소이므로 에이전트 정의에 API 키·비밀번호 같은 민감정보를 넣지 말 것
- 어느 기기에서 만들어도 상관없음 (폰에서 새로 만들어도 됨), 단지 이미 데스크탑에 정의해둔 게 있으면 그쪽이 편할 뿐

### 파일 형식 예시
```markdown
---
name: my-agent
description: 이 에이전트를 언제 호출해야 하는지 설명
tools: Read, Bash, Grep
model: sonnet
---

(에이전트 시스템 프롬프트 본문)
```
